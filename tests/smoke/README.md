# Smoke Tests

This directory contains the runtime smoke-test procedure for The Hive Skill.

## Contents

- `fixtures/` — Test fixtures for smoke scenarios

## Running Smoke Tests

Smoke tests require a live runtime with tool access. They cannot be run as
part of offline CI.

See [RELEASE_VALIDATION.md](../../docs/RELEASE_VALIDATION.md) for the full
procedure and report format.

## Fixtures

The `fixtures/` directory contains controlled test scenarios for smoke
testing. These provide known starting states for each scenario.
